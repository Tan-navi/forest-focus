# 🌿 Forest Focus

Forest Focus is a simple and responsive To-Do List web application designed to help users organize daily tasks with a clean, nature-inspired interface.

## ✨ Features

- ➕ Add new tasks
- ✅ Mark tasks as completed
- ✏️ Edit existing tasks
- 🗑️ Delete tasks
- 📊 Remaining task counter
- 💾 Local Storage support (tasks remain after refreshing)
- 📱 Responsive design for desktop and mobile devices

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Local Storage API

## 📂 Project Structure

```
Forest-Focus/
│── index.html
│── style.css
│── script.js
└── README.md
```

## 🚀 How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in your web browser.

## 📸 Project Overview

Forest Focus provides a distraction-free environment for managing daily tasks. Users can create, edit, complete, and remove tasks while all data is automatically saved using the browser's Local Storage.
